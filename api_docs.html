Ambiguous or high-risk documents escalate to a human reviewer with full context.

POST /documents/text Ingest as plain text — great for demos ▼
Submit any document as plain text. The pipeline runs synchronously and returns the full result including classification, extracted fields, validation outcome, routing decision, and complete audit trail.

Human-in-the-Loop Review
Escalation queue for low-confidence and high-risk documents
GET /review/ All documents pending review ▼
Returns all escalated documents with SLA deadlines. In production this feeds the Power Apps review portal. Includes confidence scores, escalation reasons, validation errors, and editable extracted fields.
Try it
POST /review/{doc_id}/resolve Submit human correction and re-process ▼
Human submits corrections. The document re-enters the pipeline at the validation step (skipping re-classification since the human has confirmed the type). Confidence is set to 1.0 — full human override.
